# Calculator-App

This project is a modern web-based calculator application with a dark-themed user interface. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

The calculator features:

Interactive circular buttons
Color-coded operations (orange for operators, blue for numbers)
Real-time display updates
Error handling (e.g., division by zero)
Responsive and visually appealing layout

It is designed to provide both functionality and an engaging user experience.
